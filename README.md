# AQI-Geospatial-Analysis-System-Data-Analysis-Project-

This project analyzes Air Quality Index (AQI) data across major Indian cities and visualizes spatial pollution patterns using an interactive geospatial map. The goal was to explore how data analysis and geospatial visualization can help identify pollution hotspots and understand environmental trends across regions.

The system processes AQI datasets, performs basic statistical analysis, and generates interactive heatmaps and location-based insights to highlight areas with poor air quality.
